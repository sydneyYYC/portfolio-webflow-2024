Hot Wax Music App

Custom React music player connected through the Spotify API. Pause, play and skip to the next song.  

Exploring by a new personal passion

‍

Hot Wax Music is a project that mixes development with my personal interest in collecting vinyl records. In order to make an app that feels like playing record digitally I decided to use the Spotify API to access a wide library of music.

‍

1. Strategic Discovery & research

2. Application design

3. Development

4. Deployment

‍

‍

Strategic discovery & research

‍

The inspiration for this project came from a desire to improve my web development skills and a passion for listening to and collecting vinyl records.

The initial idea of this project was to create an app that could capture a specific feeling of listening to a new vinyl. I wanted to invoke a certain feeling of 'coolness' with the visuals and have the app work functionally by playing a song - or album.

This desire informed the rest of the design and development of this app.

‍

Application Design

‍

Application design began by collecting a lot of reference photos to other music player apps, and record players. This would give me a base to build off of for the final look.

At one point the application was built for mobile, but in the end it became a desktop app.

‍

Development

‍

This app is built in React and connected to the Spotify API to play music. The app connects to the individuals account, meaning users need to log in to access music services. This is a safety feature from Spotify, and therefore I had no choice but to force users to log in.

‍

The app includes a set of controls at the bottom of the page which allows users to pause, play and skip the currently playing song. There is also a display of what song is playing, the artist, and the cover art of that song.

All of this data is being imported by API fetches to the Spotify API.

To set up these API fetches I used Postman to test the Spotify API and build my frontend functions around the data I was receiving.

‍

Deployment

‍

The final app was maintained through Github and hosted through Netlify. View the live website down below.
